# ATM Management System

A simple console-based ATM simulator written in C++.

## Features

- User authentication with account number and PIN
- Check account balance
- Withdraw cash
- View user details
- Update mobile number

## Prerequisites

- C++ compiler (e.g., g++, Visual Studio)
- Windows OS (due to use of conio.h and _getch())

## Compilation

Compile the code using a C++ compiler:

```
g++ atm.cpp -o atm
```

## Usage

Run the executable:

```
./atm
```

Follow the on-screen prompts to interact with the ATM.

Default user details:
- Account No: 987654321
- Name: Nitish
- PIN: 1234
- Balance: 50000
- Mobile: 9056152704
